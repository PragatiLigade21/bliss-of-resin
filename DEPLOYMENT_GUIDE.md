# Bliss of Resin - Deploy & Run Guide\n\n## ЁЯЪА Getting Started\n\n### Prerequisites\n- Node.js (v16 or higher)\n- npm or yarn\n- MongoDB (for backend)\n- Your backend server running on http://localhost:5000\n\n---\n\n## ЁЯУж Installation\n\n### 1. Install Dependencies\n```bash\ncd client\nnpm install\n```\n\n### 2. Verify Build\n```bash\nnpm run build\n```\nYou should see:\n```\nтЬУ 2159 modules transformed\nbuilt in X.XXs\n```\n\n---\n\n## ЁЯТ╗ Development\n\n### Start Development Server\n```bash\nnpm run dev\n```\n\nThe app will be available at `http://localhost:5173`\n\n**Features:**\n- Hot Module Replacement (HMR) - instant code updates\n- Fast refresh for React components\n- Source maps for debugging\n\n### Environment Setup\nCreate a `.env.local` file (optional):\n```env\nVITE_API_URL=http://localhost:5000\n```\n\nThen update API calls if needed:\n```javascript\nconst API_URL = import.meta.env.VITE_API_URL || 'http://localhost:5000';\nfetch(`${API_URL}/api/products`)\n```\n\n---\n\n## ЁЯПЧя╕П Building for Production\n\n### Build Command\n```bash\nnpm run build\n```\n\n**Output:**\n- Creates `dist/` folder\n- Minified and optimized bundles\n- Ready for deployment\n\n**Size:**\n- Total: ~425KB (gzipped)\n- Fast load times\n- Optimized for performance\n\n### Preview Production Build\n```bash\nnpm run preview\n```\n\nPreview at `http://localhost:4173`\n\n---\n\n## ЁЯМР Deployment Options\n\n### Option 1: Vercel (Recommended for React)\n\n1. **Push to GitHub**\n```bash\ngit init\ngit add .\ngit commit -m \"Initial commit\"\ngit push origin main\n```\n\n2. **Deploy on Vercel**\n   - Go to https://vercel.com\n   - Click \"New Project\"\n   - Import your GitHub repository\n   - Configure:\n     - Framework: Vite\n     - Root Directory: client\n     - Build Command: `npm run build`\n     - Output Directory: `dist`\n   - Add Environment Variables\n   - Deploy\n\n3. **Configure API Endpoint**\n   - Update API calls to production backend\n   - Example: `https://api.blissofres.com`\n\n### Option 2: Netlify\n\n1. **Connect Repository**\n   - Go to https://netlify.com\n   - Click \"New site from Git\"\n   - Connect GitHub\n\n2. **Build Settings**\n   ```\n   Base directory: client\n   Build command: npm run build\n   Publish directory: dist\n   ```\n\n3. **Deploy**\n   - Add environment variables\n   - Deploy\n\n### Option 3: Traditional Hosting (AWS, DigitalOcean, etc.)\n\n1. **Build Locally**\n```bash\ncd client\nnpm run build\n```\n\n2. **Upload to Server**\n```bash\n# Copy dist folder contents to server\nrsync -avz dist/ user@your-server:/var/www/bliss/\n```\n\n3. **Configure Web Server**\n\n**Nginx Configuration:**\n```nginx\nserver {\n    listen 80;\n    server_name blissofres.com www.blissofres.com;\n\n    root /var/www/bliss;\n    index index.html;\n\n    location / {\n        try_files $uri /index.html;\n    }\n\n    location /api {\n        proxy_pass http://localhost:5000;\n    }\n}\n```\n\n**Apache Configuration:**\n```apache\n<VirtualHost *:80>\n    ServerName blissofres.com\n    DocumentRoot /var/www/bliss\n\n    <Directory /var/www/bliss>\n        RewriteEngine On\n        RewriteBase /\n        RewriteRule ^index\\.html$ - [L]\n        RewriteCond %{REQUEST_FILENAME} !-f\n        RewriteCond %{REQUEST_FILENAME} !-d\n        RewriteRule . /index.html [L]\n    </Directory>\n\n    ProxyPass /api http://localhost:5000\n    ProxyPassReverse /api http://localhost:5000\n</VirtualHost>\n```\n\n---\n\n## ЁЯФз Configuration for Production\n\n### 1.
